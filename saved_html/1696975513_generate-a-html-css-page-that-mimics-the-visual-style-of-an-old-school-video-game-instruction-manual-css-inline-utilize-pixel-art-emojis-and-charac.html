Instruction Manual

Controls

🎮 Move: Arrow keys

🅾️ Action: Spacebar

🔄 Reset: R key

Gameplay

🌟 Collect stars to earn points.

🔑 Find keys to unlock new levels.

❌ Avoid enemies to stay alive.

Scoring

⭐️ Star: +100 points

🔑 Key: +500 points

💀 Enemy: -200 points

Level Select

1️⃣ Level 1: Easy

2️⃣ Level 2: Medium

3️⃣ Level 3: Hard
